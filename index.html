Introduction to HTML

HTML (HyperText Markup Language) is the standard language for creating web pages. It uses elements (also called tags) to describe the structure of a page. 

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
  </body>
</html>

Basic HTML Tags

  • <h1>...</h1> – Heading (1 is biggest, 6 is smallest)
  • <p>...</p> – Paragraph
  • <a href="...">...</a> – Link
  • <img src="..." alt="..."> – Image
  • <ul><li>...</li></ul> – Unordered list
  • <ol><li>...</li></ol> – Ordered list
<h1>My Title</h1>
<p>This is a paragraph of text.</p>
<a href="https://example.com">Go to example</a>

Basic Page Structure

A common HTML page layout uses semantic tags:

  • <header> – top section (logo, navigation)
  • <nav> – main site navigation
  • <main> – main content of the page
  • <section> – grouped content
  • <footer> – bottom of the page
<header>...</header>
<nav>...</nav>
<main>
  <section>...</section>
</main>
<footer>...</footer>

HTML Forms

Forms let users send data to your website.

<form action="/submit" method="post">
  <label>
    Name:
    <input type="text" name="name">
  </label>
  <br>
  <label>
    Email:
    <input type="email" name="email">
  </label>
  <br>
  <button type="submit">Send</button>
</form>
